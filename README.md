# CareerPrep - UX/UI Project
## Description
A UX/UI design for a career preparation app that helps students manage interview prep, activities, and career events. Designed in Figma and built with Glide no-code platform.

## Features
- User authentication
- Interview preparation tracking
- Event calendar
- Activity management
- User dashboard

## Technologies Used
- Figma (UI/UX Design)
- Glide (No-code Platform)
- Microsoft Excel
